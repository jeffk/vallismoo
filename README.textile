h2. About

These design documents are from the <a href="http://www.ghostmoo.org/">GhostMOO</a> <a href="http://fazigu.org/~quinn/ghost/files/ghostcore/">GhostCore</a> based MOO named VallisMOO.  Most of these features were never implemented, but they're archived here for any interested parties.

h2. Creators

The concept for VallisMOO was created by <a href="http://www.jeffkramer.com">Jeff Kramer</a>, with input from some compatriots I can't exactly remember.
